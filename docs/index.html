PyQuiz

PyQuiz is a console-based software testing project built in Python. The project demonstrates modular design, dependency injection, unit testing, mocking, GitHub workflow, and testing documentation.

View Repository Open Test Report

Project Focus

  • Console quiz application in Python
  • Unit testing with unittest
  • Mocking with unittest.mock
  • Structured modules in src/ and tests/

Main Modules

  • QuizEngine
  • QuestionRepository
  • QuestionOrderService
  • AnswerParser
  • ScoreService

How It Is Tested

  • Automated unit tests for services and quiz flow
  • Manual run of python -m src.main
  • Input validation and score handling verification
  • Randomized question order tested in isolation
